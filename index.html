About

Written by Samuel Reed, 2023.

Inspired by the game I sank far too much time into.

Written without any UI libraries, using HTML, SCSS, and Typescript.

This project was a fun experiment in writing OOP code instead of my normal functional code.

How To Play

Clear the mines by clicking on all the squares which are NOT mines.

Right click to set a flag. A flagged square will not be clickable, stopping you from accidentally detonating a mine.

Middle click a number to open up all surrounding (non-flagged) squares.
